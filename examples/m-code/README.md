# M code examples

Standalone Power Query M scripts, reusable snippets, and related examples.

## Gists

Some examples are kept as GitHub Gists so existing links continue to work and the original version history remains available.

- [Extended Date table | Creates an ISO-8601 type calendar](https://gist.github.com/m-dekorte/12b53faee9cc1a616fa23f15b1b4a173)
- [fillRightOrLeft | fill values right (or left) with options](https://gist.github.com/m-dekorte/56b996be4f51a6eaab6132e1e46db30d)
- [errorReport | Creates a record with error count and detailed error reporting table](https://gist.github.com/m-dekorte/37be84d191873ec6cd9b918f1d32dfb0)
- [profileWithErrorCount | Creates a table profile with statistics, incl an error count for all columns](https://gist.github.com/m-dekorte/eb2bb9947b57c8a68c88e64436047340)
- [errorReplacer | Substitute errors in all or listed scanColumns with a specified value](https://gist.github.com/m-dekorte/f35837d4f70af1fcfa10c2ee07127deb)
